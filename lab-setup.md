## Cisco setup
1. Change defualt password

2. Enable ssh  ( from console ) 
3. Enable dhcp  
3.1 ssh switch and run   
```
configure terminal
interface vlan1
ip add 10.0.0.1 255.255.255.0
```
3.2 from switch UI enable DHCP  

4. Create DHCP network pool 
range should be 10.0.0.100 - 10.0.0.250
